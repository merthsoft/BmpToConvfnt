# BmpToConvfnt

Creates a Convfnt ascii file from an image file.

```
Usage: bmptoconvfnt imagefile width height startcodepoint [numglyphs]
  imagefile       - Path to the BMP image file
  width          - Width of each glyph in pixels
  height         - Height of each glyph in pixels
  startcodepoint - Starting code point (e.g., 32 for space, 65 for 'A')
  numglyphs      - (Optional) Number of glyphs to read from the image
```
